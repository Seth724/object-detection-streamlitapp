## 📅 5-Day Master Plan

### 🗓️ Day 1 — Object Detection Basics & First Run
- Understand object detection fundamentals (bounding boxes, confidence scores, IoU)
- Run your first inference using a pre-trained model

### 🗓️ Day 2 — Real-Time Video & Performance Analysis
- Integrate real-time webcam feeds
- Monitor FPS and latency to analyze performance

### 🗓️ Day 3 — Model Fine-Tuning & mAP Evaluation
- Fine-tune models on a custom dataset
- Evaluate using mAP (mean Average Precision) to boost accuracy

### 🗓️ Day 4 — Optimization & Deployment
- Optimize model size and inference speed
- Export and deploy your model with Streamlit

### 🗓️ Day 5 — App Build & Portfolio Showcase
- Wrap up everything into a Streamlit app
- Polish the UI and publish your project to your portfolio ✨
